Test para probar flags del printf.

Copia en esta carpeta tu 'libftprintf.a' y ejecuta el test que quieras con el comendo bash:

$> bash str_test.sh
$> bash nb_test.sh
$> bash hexa_test.sh
$> bash mem_test.sh

El test.sh no hace nada, esta ahi para que puedas hacer tus propias pruebas.
Modifica el archivo 'test.c' ubicado en la carpeta my_test/ y ejecuta el comando 'bash test.sh'
